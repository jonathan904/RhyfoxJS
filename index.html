    What is RhyfoxJS?

    RhyfoxJS is a open source platform that allow integrate multiple plugins written in JavaScript for analyze web aplications and runs without need the Web Browser. RhyfoxJS is a utility based in PhantomJS and CasperJS. With RhyfoxJs you could common tasks such as:

    • Search Broken Links.
    • Evaluate Time Load Resources.
    • Search Https Security Errors.
    • Analyze redirects resources .
    • Search Broken Resources.
    • Note: RhyfoxJS latest released Version is 1.0.0.

    Get Started

    This example explain how you can create your plugins and include in RhyfoxJS. Get support. Checkout the code at Github. Follow @RhyFoxJS on Twitter.

    1. Create Plugin Folder

      Create a folder into plugins folder of RhyfoxJS project. 
         $ cd ~/RhyfoxJS/plugins/
   $ mkdir /MyPlugin
   $ cd /MyPlugin

    2. Create Plugin File

      The plugin file name must match the plugin folder name. If your plugin require a config file must match the folder name and add the prefix 'config_' in the beginning of the config file name. This config file must contain a JavaScript Object (JSON), this is an optional file.

      MyPlugin.js

      MyPlugin = function(ObjConfigPlugin) { // config JSON 
                  this.url = ObjConfigPlugin.url;
                  this.run = function() {
                                console.log('Hello World!!');
                                this.api.finish(); //finish plugin
                             }
           }

      config_MyPlugin.json

      var config_MyPlugin = { // optional file
                        name: 'NoFound404Plugin',
                        urls: ['URL_Webpage','URL_Webpage2']
                      }

    3. Run Plugin

      RhyFoxJS executes the plugins of automatically for analyze any web aplication. For run the plugin run it: 
      $ cd ../../src/
$ phantomjs bootstrap.js

    Installation

    Once PhantomJS's installed on your machine, you should obtain something like this:

    $ phantomjs --version
1.5.0

    Now just follow these instructions:

    $ git clone git@github.com:jonathan904/RhyfoxJS.git
$ cd RhyfoxJS

    You are now ready to create plugins and integrates with RhyfoxJS!
